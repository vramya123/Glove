* Glove pretrained model used in this note can be downloaded as follows:-

Goto https://github.com/stanfordnlp/GloVe

Download the twitter pretained model ie, "Twitter (2B tweets, 27B tokens, 1.2M vocab, uncased, 200d vectors, 1.42 GB download): glove.twitter.27B.zip"

Unzip the download and store the "glove.twitter.27B.100d.txt" in a local path and use in the model.

* Please change the dataset path and the pretained model path in the notebook before running the note.
